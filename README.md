# dji-tello-playground

# Dependencies

- Python > 3.11

# Setup

We need to connect to the drone directly with WiFi with our computer before running any scripts.

![WiFiConnection](./docs/images/trello_wifi.png)

# Exercises

Find all the scripts in the `src` folder.

1. Manual control with smart phone
2. Simple take off
3. Follow with face detection
4. Navigate route (bonus for fastest lap)
5. Orchistrate swarm
6. Do stunts
7. Make deliveries
